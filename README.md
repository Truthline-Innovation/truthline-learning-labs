# TruthLine Learning Labs

This repository is the practical learning space for members of
**TruthLine Innovations**.

Here, Computer Science students collaborate, practice, experiment,
and build real-world skills through hands-on coding.

---

## Learning Tracks
- Web Development (HTML, CSS, JavaScript)
- Backend Development (PHP, MySQL)
- Mobile Development (Flutter)
- Git & GitHub
- Mini Projects

---

## How to Contribute
1. Join the TruthLine Innovations GitHub organization
2. Pick a learning track
3. Create a folder with your name or topic
4. Push changes to a branch
5. Open a Pull Request
