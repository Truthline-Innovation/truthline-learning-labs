# Backend Task 001 – PHP & MySQL Basics

## Objective
Build a simple user registration system using PHP and MySQL.

## Requirements
Each member should create a folder with their full name inside this directory.

Inside your folder, include:
- index.php
- register.php
- db.php
- README.md

## Features
- HTML form (name, email, password)
- PHP form handling
- MySQL database connection
- Basic validation

## Rules
- Do not copy another member’s solution
- Keep credentials out of public commits
- Comment your code where necessary

## Bonus (Optional)
- Password hashing
- Simple success/error message
